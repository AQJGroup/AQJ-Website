Software Ecosystem

AQJ Group uses a curated software stack for modelling, coordination, simulation and delivery. We combine industry-standard platforms with automation tools to deliver accurate models, clash-free coordination and operationally-ready asset data.

Primary BIM & Coordination Platforms

Autodesk Revit

Our primary modelling tool for architectural, structural and MEP design. We author families, templates and perform LOD-based modelling in Revit.

Navisworks Manage & Clash Detective

Federation, clash detection, model review and coordination workflows — Navisworks is used for multi-discipline coordination and issue tracking.

Autodesk BIM 360 / Autodesk Construction Cloud

Document & model management, cloud-based coordination, markups and controlled collaboration across project teams.

Documentation & CAD

AutoCAD

Drawing production, 2D documentation and interoperability with older CAD workflows and consultants.

Bluebeam Revu

PDF markup, QA/QC review, punchlists and digital collaboration for document-heavy workflows.

Microsoft Office / Excel / Power BI

Project reporting, data analysis, and dashboards for handover and O&M teams (Power BI for dashboards).

Visualization & Rendering

Twinmotion

Fast, real-time visualisation for client presentations, VR-ready exports and animation of construction sequences.

V-Ray / Corona

Photoreal renders for high-fidelity imagery used in marketing and stakeholder sign-off packages.

Unreal Engine / Datasmith

Real-time engines for immersive walkthroughs, complex visualization and advanced simulation use-cases.

Analysis, Simulation & Automation

Primavera P6 / Microsoft Project

Schedule integration for 4D simulations and construction sequencing analysis.

Dynamo & Python

Parametric automation, batch processing and data extraction — Dynamo for Revit workflows and Python for data pipelines.

BIM Query & Data Tools

Tools like Solibri, BIMcollab, and custom scripts for model checking, data validation and rule-based QA.

Cloud, CI & Integration

APIs & Integrations

Integration with BIM 360/ACC APIs, webhooks and scripted exports to connect models with FM and reporting systems.

Docker / CI

Containerised processing pipelines for repeatable data transforms, CI for validation and automated exports.

Cloud Platforms (Azure / AWS)

Cloud-hosted services for dashboards, storage and server-side model processing when needed.

Plugins, Utilities & File Formats

IFC / COBie / gbXML

Open exchange formats we validate and use for interoperability between platforms and for handover packages.

Revit Add-ins & Utilities

Custom add-ins and community tools to speed modelling, naming and parameter management.

Reporting & Dashboards

Power BI, custom web dashboards and automated exports for stakeholders and FM teams.

Licensing & Training Support

We help clients select appropriate software licensing and deliver role-based training to ensure teams can use tools effectively. From admin & IT setup to hands-on user sessions, our training covers modelling best practices, coordination workflows and automation.
